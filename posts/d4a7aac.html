玩转虚拟机之为PC安装ESXI
发表于2021-03-17|更新于2021-03-18|硬件相关ESXI
|字数总计:2.7k|阅读时长:8分钟|阅读量:|评论数:

前言

  ESXI是什么？
  答: 按照官网的意思就是一个虚拟机管理程序，通过将计算机的软件与其硬件分离，支持创建和管理虚拟机。

  我为什么要用ESXI？
  答: 第一、我想找一个专门用于虚拟机创建和管理的平台，我在用的群晖系统和Unraid只是兼有这个功能；
    第二、其他的用过Proxmox、Hyper-V，前者做集群没有ESXI稳定，后者远程管理用的是Windows远程桌面；
    所以，我使用ESXI是因为集群稳定且远程管理更加喜欢。

准备工作

硬件

  • 一台电脑 (能够正常开机且接口正常使用，以及有一块硬盘)
  • U盘 (已经做成PE盘)

软件

  • ESXI镜像文件(点击下载密码：4jj0)
  • ESXI网卡驱动文件以及注入驱动工具(点击下载密码：lx26)
  • ESXI注册机(点击下载密码：gcq5)
  • 写盘工具(点击下载密码：7d9j)

开始

下载镜像

  虽然上面已经给出了几个ESXI版本的镜像供大家下载，但是我这边还是教一下大家怎么去官网下载。

  1. 进入官方网站：https://www.vmware.com/cn.html ，点击左上方的下载选项；

  2. 在出现的下拉菜单中点击 产品下载 中的 vSphere；

  3. 出现下图下载 VMware vSphere的界面，此时显示的是ESXI7.0的下载；

  4. 点击选择版本中的下拉选项框，选择ESXI 6.7，等待下面产品中出现 VMware vSphere Hypervisor (ESXi) 6.7U3b；

  5. 点击右边的 转至下载；

  6. 等待页面跳转出现下图界面，可以看到下载的产品版本为 6.7.0U3B，点击 VMware vSphere Hypervisor (ESXi ISO) image (Includes VMware Tools) 右边的立即下载，如果此时还未登陆会跳转到登录页面，有账号直接登录没有账号按步骤注册账号并登录；


  7. 登录后会跳转到下图界面，选择 许可证和下载 ，然后再点击注册（6.0-6.7版本注册码：0U0QJ-FR1EP-KZQN9-J1C74-23P5R ）；

  8. 注册完信息后就可以真正下载了。

  9. 该下载方法可以在60天内免费下载试用版，过时的话需要重新注册新账号再按照上述方法操作。

安装ESXI

方法一

  1. 拿出准备好的U盘，用工具安装PE系统到U盘（这里推荐一下大白菜PE，简单易用，虽然有些小问题），然后把下载好的ESXI镜像放进根目录下的 IMG （没有就创建一个）；
    （ps：U盘启动盘制作教程请看这“大白菜超级u盘装机工具”）

  2. 把制作好的U盘插到你需要装ESXI的电脑上，开机选择U盘启动，进入PE系统，通过类似DiskGenius这样的磁盘管理软件，将安装ESXI的硬盘删除所有分区，然后保存更改；

  3. 接着，重启电脑，同样是进入U盘的PE引导，这次选择非UEFI的方式，然后出现下面的界面，选择运行自定义镜像，等待系统扫描U盘的镜像，然后选择我们放入到U盘的ESXI镜像，然后按回车键就可以进入ESXI的安装界面了；

  4. 出现图1界面，按回车键选择第一个，等待安装程序的加载（该过程的等待时间受配置等一些因素影响，时间长短不一，请耐心等待），出现图2带有 Welcome to xxxx 的界面即可开始安装，先按回车键；


    若出现 Shutting down xxxxx 无法正常引导内核错误的界面，解决方法如下

    1、启动到如下安装选择界面，按 Tab 键；


    2、添加 ignoreHeadless = TRUE 这条参数；


    3、接着按回车键并重启电脑。

    若出现 Not Network Adapters 界面，解决方法如下

    1、出现下图所示界面，则便是ESXI无法识别网卡；


    2、先查询一下主板网卡型号，然后到VMware官方提供了硬件兼容列表查询；


    3、鉴于驱动查找麻烦，我在上面软件已经打包好了部分常见过的网卡驱动以及驱动注入工具，下载使用即可；
    4、下载 ESXI网卡驱动及安装程序.zip ，解压并打开，找到 ESXi-Customizer-v2.7.2.exe.zip 解压并打开，然后点双击 ESXi-Customizer-v2.7.2.exe 安装，接着打开 ESXi-Customizer-v2.7.2 文件夹，双击打开 ESXi-Customizer.cmd 文件，出现下图界面。


    目前已有的驱动如下：


    5、如果嫌麻烦请直接使用网盘中已经集成驱动的两个镜像文件：ESXI_6.7集成驱动版 或 ESXi-5.5-Custom.iso 。

  5. 出现下图1界面请按 F11 同意安装，接着等待扫描硬盘，出现图2界面，选择你安装ESXI的硬盘，接着按回车键；


  6. 接下来出现输入密码的界面，按照 大小写字母加符号且八位以上 的密码规则输入密码，密码输入完成后按回车键进入下一步；

  7. 出现图1界面，按按 F11 开始安装ESXI，等待一段时间安装完成，出现图3界面，按回车键重启，这时即可拔出U盘；



  8. 等待出现下图带有IP的界面出现，则代表ESXI安装成功。

方法二

  该方法不需要制作PE盘，只需要用写盘工具将ESXI镜像写入U盘，然后同样通过U盘启动即可，剩余的步骤与方法一中的 4、5、6、7、8 步骤相同，写入镜像方法如下：
  下载网盘中的写盘工具（四个选一个即可，我用的是 UltraISO ），打开写盘工具，点击 文件 → 打开 → 选择ESXI镜像文件 ，接着点击 工具 → 写入硬盘镜像 ，出现图3界面，先点击 格式化 ，再点击 写入 ，待写入完成，拔下U盘，再把U盘插入需要安装ESXI的电脑中，然后开始方法一中的 4、5、6、7、8 步骤。



配置ESXI

配置空密码

  在如下界面按 F2 进入ESXI设置，输入之前安装时设置的密码，进入设置选项，选择最后一个 Reset System Configuration 重置系统配置，出现弹窗，按 F11 确认重置，等待重启，回到主页面，再次按 F2 进入ESXI设置，出现弹窗，这是不输入密码直接按回车键即可设置为空密码。

  该方法仅建议平时家用ESXI时先输入密码麻烦时使用，但我个人并不建议设置空密码。



配置ESXI后台地址及网关

  按 F2 进入ESXI设置，选择第三项 Configure Managenent Network ，进入设置，选择 IPv4 Configuration ，出现设置弹窗，移动到 Set static IPv4 address and network configuration ，然后按空格键选中，接下来在下面填写IP地址、子网掩码和网关，例如我的是 192.168.12.222 、 255.255.255.0 、 192.168.12.1 ，具体请按照你自己路由器相关IP来设置，完成后按回车键，然后按 ESC 键出现弹窗，接着按 Y 选择保存设置。

  接下来一直按 ESC 键退出到主界面查看IP地址是否变为我们修改后的IP地址，如果地址为我们设置的IP，则代表设置成功，如下图所示。




开启ESXI的SSH和Shell

  按 F2 进入ESXI设置，选择第八项 Troubleshoot ing Mode Options ，进入设置，可以看到ESXI的SSH和Shell都是出于关闭状态（disabled），我们分别选中SSH和Shell的选项，然后按回车键即可开启两个功能，接着按 ESC 键退出该界面即可保存成功，如下图所示。


激活ESXI为永久使用

  配置完后台地址等设置，在同一内网的电脑打开浏览器，输入前面设置的IP地址即可进入web页面，输入账号 root 以及之前设置的密码（如果设置为空密码就不需要输入密码），点击登录即可进入ESXI后台页面。

  看到如图1界面，点击左上角 导航器 → 管理 → 许可 → 分配许可证 ，输入密钥 0U0QJ-FR1EP-KZQN9-J1C74-23P5R ，点击检查许可证 → 分配许可证，如果过期时间如图所示从确切的日期变成从不，则代表成功。


若安装的是ESXI 6.5以下的版本，请按照以下方法登录后台

1、安装 VMware vSphere Client (点击下载密码：a9op)
 （仅提供版本号5.5的客户端，其他版本客户端请点击跳转前往下载）
2、打开安装好的 VMware vSphere Client ，输入前面设置好的IP、账号root以及密码。


3、进入如下图界面，点击 配置 → 已获许可功能 → 编辑 ，选中 向此主机分配许可证（我这个已经激活所以演示效果是这个，不过新安装的也差不多），点击 输入密钥 ，弹出输入密钥的弹窗；


4、下载并打开上面给出的网盘中的ESXI注册机，选择对应的产品，点击 Generate 生成对应的激活许可，然后把激活许可复制粘贴到上面的弹窗中，再点击确定即可激活许可，看到过期从确切的日期变成从不，则代表成功。

最后

  看到这里，则代表本次关于ESXI的安装以及基础的配置的相关教程已经完成了，接下来会有进阶版的教程，更新时间待定，毕竟这类文章写出来要一步步重新搞一次，还要截图，比较麻烦。

  最后祝大家完美安装ESXI，成功用上想要的功能，那么再见👋。
